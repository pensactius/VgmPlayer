Detailed Description

Audio chip Ym-2612 class.

YM2612 test code for AVR.

This program is a simple test code for the YM2612 FM sound chip using an AVR ATmega328p mcu. This program configure the YM2612 to sound like a "grand piano" and play note on / note off in loop. For more informations about wiring please see: http://en.wikipedia.org/wiki/Yamaha_YM2612 For more informations about YM2612 registers please see: http://www.smspower.org/maxim/Documents/YM2612

Remarks
This test code is heavly based on Furrtek's YM2612 test code. Big thanks Furrtek for the help !
Warning
This test code is made to run on an ATmega328/ATmega168 mcu with a 16MHz external crystal.
